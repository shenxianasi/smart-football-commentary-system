# 蓝心AI足球解说系统 - 前端界面

## 项目概述

这是一个基于HTML、CSS和JavaScript的前端界面，为蓝心AI足球解说系统提供用户交互，实现了上传视频、选择解说语言和风格、生成解说视频等功能。

## 功能特点

- **视觉效果**：采用利物浦主题风格配色，具有现代化的UI设计和流畅的动画效果
- **遮罩层**：进入系统前显示精美的遮罩层，点击或向下滑动可进入主界面
- **视频上传**：支持多种视频格式上传（MP4、AVI、MOV、MKV、FLV）
- **多语言支持**：支持汉语、英语、法语、德语、俄语、阿拉伯语、西班牙语、韩语等多种语言的解说
- **多风格选择**：支持默认、幽默、激情、专业等多种解说风格
- **视频预览**：上传视频后可实时预览，生成解说后可播放带解说的视频
- **全屏播放**：支持全屏播放生成的解说视频

## 技术栈

- **前端**：HTML5、CSS3、JavaScript
- **后端**：Python Flask
- **响应式设计**：支持桌面和移动设备

## 目录结构

```
web_frontend/

├── README.md         # 项目说明文档
```

## 快速开始

### 前提条件

- 已安装Python 3.6或更高版本
- 已安装Flask库
- 已安装原项目中的所有依赖（用于AI解说生成）

### 安装依赖

```bash
pip install flask
```

### 运行服务器

1. 进入web_frontend目录

```bash
cd web_frontend
```

2. 启动Flask服务器

```bash
python server.py
```

3. 打开浏览器，访问以下地址

```
http://localhost:5000
```

## 使用方法

1. **进入系统**：点击遮罩层或向下滑动进入主界面
2. **上传视频**：点击上传视频按钮，选择一个足球比赛视频文件
3. **选择语言**：从下拉菜单中选择希望的解说语言
4. **选择风格**：从下拉菜单中选择希望的解说风格
5. **生成解说**：点击"开始生成解说"按钮，等待系统处理视频
6. **播放视频**：生成完成后，系统会自动播放带解说的视频
7. **全屏播放**：点击"全屏播放"按钮可全屏观看视频

## 注意事项

- 视频处理时间取决于视频长度和您的计算机性能
- 为获得最佳体验，建议使用Chrome、Firefox、Edge等现代浏览器
- 当前版本的后端服务仅作为演示，实际应用中需要将其与原项目的AI解说生成系统集成

## 集成说明

web_frontend已经与项目的其他部分完全集成。`server.py`文件中的`generate_commentary`函数已经实现了与原项目`run_AIGC.py`脚本的对接逻辑。

主要集成点包括：
1. 上传的视频文件会自动复制到项目根目录的`input_videos`文件夹
2. 设置环境变量传递语言和风格参数
3. 调用`run_AIGC.py`脚本执行完整的解说生成流程
4. 查找生成的最终视频文件并复制到web_frontend的`outputs`文件夹
5. 提供视频播放链接供前端展示
